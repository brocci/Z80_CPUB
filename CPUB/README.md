## 注意点

- SBC8080ルーズキットと完全互換ではありません。
  - DMAは行なえません。(#BUSREQ/#BUSACKに対応していない)
  - DMAに伴う#IOR/Wや#MEMR/Wのハイインピーダンス対応を実施していません。
