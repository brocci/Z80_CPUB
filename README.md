# Z80マイコンボード

SBC8080バスにつながるマイコンボードや周辺ボードのデータ類です。
CPUボード+SBC8080 SUBボードのセットなどでコンピューターとして動作します。

## サブディレクトリ

以下のディレクトリがあります。

|ディレクトリ|概要|
|:--|:--|
|CPUB|Z80 CPUボード|
|RAMB|128kBバンクメモリーボード|
|IOB|8255/8251ボード|
